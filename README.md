# INSTALL

### Command 
`npm install -g yarn; ls; cd src; yarn install; cd ..; npm run lint -- --fix; npm install --save @fortawesome/fontawesome-free`



# BUILD

### Command to Run Developer mode
`yarn dev`

### Command to Build App
`yarn build`
